# Woodbox Demo

Самостоятельная интерактивная демонстрация современного магазина предметной мастерской Woodbox.

## Запуск

```bash
pnpm install
pnpm dev
```

Проверка: `pnpm verify`. Health endpoint: `GET /api/health`.

Данные каталога локальные и типизированные. Исходные карточки и фотографии получены из публичного магазина `livemaster.ru/woodbox`; в интерфейс отобраны 10 изделий. Демо не принимает оплату, не отправляет формы и не использует backend или базу данных.
